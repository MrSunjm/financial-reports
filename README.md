# 📊 金融市场分析报告库

## 🎯 项目简介
每日自动生成的金融市场分析报告，包含：
- 国际金融市场分析（美股、A股、港股）
- 加密货币市场分析（BTC、ETH、BNB等）
- 技术指标计算和风险评估
- 投资建议和策略

## 📅 更新频率
- 每日 08:00（北京时间）自动更新
- 报告保留最近100份
- 自动清理旧报告

## 🌐 访问方式
1. **GitHub Pages**: https://[用户名].github.io/financial-reports/
2. **直接访问**: 点击 reports/ 目录下的HTML文件

## 📱 移动端支持
- 完全响应式设计
- 支持所有现代浏览器
- 手机、平板、电脑均可访问

## 🔧 技术栈
- Python 数据收集和分析
- HTML/CSS/JavaScript 报告生成
- GitHub Pages 部署
- 自动化定时任务
